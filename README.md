# PDF注釈エディタ

既存のPDFに **テキスト・ペン・ハイライト・図形・線/矢印・画像** を重ねて書き込み、注釈を焼き込んだPDFとして保存できる、ブラウザ完結のエディタです。

ファイルはすべて**端末内（ブラウザ内）だけで処理**され、どこのサーバーにもアップロードされません。完全オフラインで動作します。

## 主な機能

- **PDFを開く** — ファイル選択 or ドラッグ&ドロップ。複数ページを縦スクロールで表示
- **注釈ツール**
  - テキスト（日本語OK。クリックで配置、ダブルクリックで再編集）
  - ペン（フリーハンド）
  - ハイライト（蛍光ペン）
  - 直線 / 矢印
  - 四角形 / 楕円（塗りつぶし切替）
  - 画像・サインの挿入
- **編集** — 選択 / 移動 / 四隅リサイズ / 端点ドラッグ、色・太さ・文字サイズ・塗り変更、削除
- **元に戻す / やり直し**（⌘Z / ⌘⇧Z）、ズーム
- **保存** — 注釈をページにラスタライズして焼き込み、新しいPDFをダウンロード（`〜_注釈.pdf`）

### キーボードショートカット

| キー | 機能 |
| --- | --- |
| `V` 選択 / `T` テキスト / `P` ペン / `H` ハイライト | ツール切替 |
| `L` 直線 / `A` 矢印 / `R` 四角 / `O` 楕円 | ツール切替 |
| `Delete` / `Backspace` | 選択中の注釈を削除 |
| `⌘Z` / `⌘⇧Z` | 元に戻す / やり直し |

## 技術スタック

- Vite + React + TypeScript + Tailwind CSS + zustand
- [pdf.js](https://mozilla.github.io/pdf.js/)（`pdfjs-dist`）— ページの表示
- [pdf-lib](https://pdf-lib.js.org/) — 注釈の焼き込み・書き出し

## しくみ

注釈は「PDFポイント・左上原点」の座標系で SVG として保持します。画面表示と書き出しで
**同じ `AnnotationShape` コンポーネント**を使うため、見た目が完全に一致します。

書き出し時は、注釈レイヤーの SVG を `renderToStaticMarkup` で文字列化 → 画像にラスタライズ →
`pdf-lib` で各ページに重ねて描画します。テキストはシステムフォントで描かれるため、
**日本語フォントを別途埋め込まなくても日本語の注釈をそのまま保存**できます
（ページの `/Rotate` 回転にも対応）。

## 開発

```bash
npm install
npm run dev        # 開発サーバー（http://localhost:5175）
npm run build      # 本番ビルド（dist/）
npm run typecheck  # 型チェック
```

`public/sample.pdf` は動作確認用のサンプルです（トップ画面の「サンプルPDFで試す」から開けます）。

## プライバシー

PDF・画像・注釈の内容は一切ネットワークに送信されません。すべてブラウザ内で完結します。
